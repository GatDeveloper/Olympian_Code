Intruções divinas

Olympian code é uma sequência de desafios sobre linguagens de desenvolvimento e programação web.

Os desafios são estruturas de sites que devem ser montados respeitando diretrizes de acessibilidade, usabilidade e responsividade.

Ao escolher HTML por exemplo, o jogador verá um desafio aleatório de estrutura para montar na linguagem e logo abaixo um Patch (Sorte) que dará alguma vantagem no desenvolvimento ou um Bug (revés) que dará alguma dificuldade. Ao escolher outras linguagens terá desafios que complementam o HTML.
